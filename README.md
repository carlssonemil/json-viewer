# JSON Viewer

A minimal JSON viewer built with Vue 3. Paste JSON in the textfield and explore it as a collapsible tree.

## Features

- Instant JSON parsing and rendering
- Collapsible tree structure for easy navigation
- Error feedback for invalid JSON
- Clean and simple user interface
- Fullscreen mode for distraction-free viewing
- Keyboard shortcuts for enhanced usability (<kbd>F</kbd> to toggle fullscreen, <kbd>Esc</kbd> to exit)

## Tech

- [Vue 3](https://vuejs.org/) for building the user interface
- [Vite](https://vitejs.dev/) for build tooling
- [Tailwind CSS](https://tailwindcss.com/) for styling
- [Tippy.js](https://atomiks.github.io/tippyjs/) for tooltips
- [Lucide Icons](https://lucide.dev/) for icons
- [Vercel](https://vercel.com/) for hosting
- [Plausible](https://plausible.io/) for [privacy friendly](https://plausible.io/privacy-focused-web-analytics) analytics

## Setup

```bash
yarn install
yarn dev
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| <kbd>F</kbd> | Toggle fullscreen mode |
| <kbd>Esc</kbd> | Exit fullscreen mode |
